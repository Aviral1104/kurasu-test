Kurasu upgrade plan — Solo Leveling features, animations, fonts, avatars and blog import

Solo Leveling UI features to add
System awakening screen Onboarding

First-time users get a dramatic full-screen black overlay — white text types out "You have been selected by the System." Then their stats appear one by one. Sets the tone immediately.

  • Typewriter animation on first login
  • "A new Hunter has awakened" message
  • Rank assigned: E-Rank by default
  • Fade into dashboard after 3 seconds
Rank-up ceremony Animation

When a user finishes a course (clears a dungeon), trigger a full-screen "Arise" moment — dark overlay, rank badge scales up, purple particle burst, sound optional.

  • Black overlay fades in on course complete
  • Old rank fades out, new rank scales in
  • Particle effect (CSS only, no library)
  • "You have leveled up, Hunter" text
Stat screen panel Feature

A dedicated page styled like the Solo Leveling stat window — glass-panel aesthetic, all stats listed like RPG attributes: INT (quizzes), PER (streak), STR (videos watched).

  • INT = quiz accuracy score
  • PER = streak consistency
  • STR = total videos completed
  • AGI = avg time to complete a course
Daily quest reset animation Animation

At midnight, quests reset with a brief flash — "New quests have been issued by the System." Completed quests get a strikethrough with a green glow before disappearing.

  • Completed quest: green glow + strikethrough
  • Midnight: "System Notice" banner slides in
  • New quests fade in one by one
Shadow collection (flashcards) Feature

Rename the flashcard section to "Shadow Army." Each completed video drops a Shadow card — styled like a collectible, with the topic name and XP value shown.

  • Shadow cards stack in a collection view
  • Rare shadows for hard topics (S-Rank material)
  • "Review your shadows" = flashcard session
Gate scanner (link import) Feature

Replace "New Gate" button with a scanner animation — user pastes a URL, a purple scanning line sweeps across it, then the Gate (course) materializes with rank assigned by AI based on content difficulty.

  • Scanning animation on URL paste
  • AI assigns rank based on length + complexity
  • "Gate identified: A-Rank Dungeon detected"
Blog / website course import — how to handle it
Single-topic blog import GeeksForGeeks, MDN, etc.

User pastes one article URL. You scrape the content, AI generates a summary, key points, and a 3-question quiz. That article becomes one "chapter" in a dungeon.

Flow: 1. Paste URL → scrape with Cheerio (Node) 2. Strip HTML → clean text 3. Send to Gemini: summarize + quiz 4. Save as single-chapter course 5. User can add more chapters manually
Multi-article series import Blog series

For sites like GFG where topics have sub-pages, let the user paste a "topic root" URL (e.g. GFG's linked list page). App crawls linked sub-articles in the same topic and builds a full dungeon automatically.

Flow: 1. Paste topic root URL 2. Crawl same-domain links (max 20) 3. Filter by topic relevance (AI) 4. Order by reading sequence 5. Build multi-chapter dungeon
Font recommendations — epic but professional
Recommended font stack

You want that Solo Leveling feel — sharp, geometric, slightly futuristic — without looking like a game title screen.

HeadingsOutfit (Google Fonts) — geometric, clean, premium feel
BodyInter — the standard for serious apps, readable at any size
Rank badges / XPSpace Grotesk — techy, sharp, perfect for numbers
System messagesSyne — slightly unusual, feels like the System speaking
CSS: --font-heading: 'Outfit', sans-serif; --font-body: 'Inter', sans-serif; --font-rank: 'Space Grotesk', sans-serif; --font-system: 'Syne', sans-serif;
Typography rules for the theme

Small typographic choices make it feel premium without being loud.

  • Rank badges: letter-spacing: 0.12em, all-caps, Space Grotesk
  • XP numbers: tabular-nums, Space Grotesk, no letter spacing
  • System messages: Syne, italic, slightly muted color
  • Quest titles: Outfit medium weight, sentence case
  • Course titles: Outfit semibold, larger tracking
  • Body text: Inter regular, 14–15px, generous line-height
