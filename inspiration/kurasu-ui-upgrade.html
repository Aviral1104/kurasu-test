A complete diagnosis of what's making Kurasu feel plain — and exactly what to fix, in order of impact.

Hunter, your Command Center lacks depth. The background is flat black — it has no atmosphere. The cards have no visual weight. The typography is mixed and inconsistent. The rank badges do not inspire awe. These issues are fixable. The System has identified 8 critical upgrades.

DIAGNOSE
WHAT'S WRONG RIGHT NOW
🎨
FLAT BLACK BACKGROUND
Pure #0a0a0a black with no depth. There's no atmosphere, no sense of a world. Looks like a half-finished dark mode.
Fix → radial gradient purple aura + pixel grid overlay
🃏
CARDS HAVE NO WEIGHT
Cards float on the background with no shadow, no glow, no border accent. They could be divs from any website.
Fix → pixel box-shadow offset + top accent border + inner glow
🔤
MIXED TYPOGRAPHY
Course titles in one font, stats in another, labels in system default. No hierarchy. Feels unplanned.
Fix → 3-font system: Pixel for labels, Rajdhani for titles, Mono for data
🏅
RANK BADGE IS TOO PLAIN
"E-RANK" button looks like a browser default. It should feel like a real rank — intimidating, earned, weighted.
Fix → pixel border + offset box-shadow + rank color per tier
📊
STAT CARDS LOOK GENERIC
White numbers on dark background. No color context. XP and streak look identical in style — they should feel different.
Fix → colored top border + colored value + glow on high stats
🌀
NO MICRO-INTERACTIONS
Nothing reacts. No hover states, no XP pop on completion, no animation on quest tick. Feels static and lifeless.
Fix → hover lifts, +XP float, quest completion flash, streak pulse
🎯
QUEST ITEMS ARE BORING
Bullet list with muted dots. No pixel aesthetic, no left border accent. Could be from any todo app.
Fix → pixel dot, left border glow, mono font, hover state
🖼️
COURSE CARD THUMBNAIL IS EMPTY
Dark grey box with a generic code icon. No rank badge, no gradient, no personality. Every course looks the same.
Fix → per-rank gradient + scanline overlay + pixel rank badge

/* 4. Typewriter */
function systemSpeak(msg, el) {
  el.textContent = '';
  let i = 0;
  const type = () => {
    if (i < msg.length) { el.textContent += msg[i++]; setTimeout(type, 38); }
  };
  type();
}
